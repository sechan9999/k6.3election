# 6.3 지방선거 결과 분석 대시보드

2026년 6월 3일 대한민국 지방선거 결과를 시각화한 인터랙티브 대시보드입니다.

## 🔗 Live Demo

[https://k6-3election.vercel.app](https://k6-3election.vercel.app)

## 📊 주요 내용

- **KPI 카드**: 서울시장 초박빙(1.15%p), 평택을 연합 실패(-21.7%p), 민주당 광역·기초 의석 변화
- **초박빙 레이스**: 서울시장(60,259표), 부산 북구(1,382표) 득표율 비교
- **평택을 해부**: 민주+혁신당 합산 56.3%가 34.6%에 패배한 연합 정치 실패 시각화
- **의석 변화**: 민주당 2022→2026 광역 5→12석, 기초 63→123석
- **중대선거구 효과**: 진보당 기초의원 30명+ 당선
- **분석 교차검증**: 정해랑(국민주권 사회대개혁 전국시국회의) 진단 vs 데이터 검증

## 🛠 기술 스택

- Vanilla HTML/CSS/JS (단일 파일, 외부 의존성 없음)
- [Chart.js 4.5](https://www.chartjs.org/) — 인터랙티브 차트
- Vercel — 정적 호스팅

## 📁 파일 구조

```
index.html     # 대시보드 (자기완결형 HTML)
vercel.json    # Vercel 정적 배포 설정
```
